# MWeb Claude-Like 主题 v2.0 ✨

基于 Anthropic Claude 设计语言的 MWeb 主题集合 - **深度优化版**

完美适配 MWeb 4 最新版本，融合 Claude 的优雅与专业感。

## 主题列表

### 1. Claude-Like Light (亮色主题) ☀️
- **文件名**: `mweb-claude-light.mwebtheme`
- **配色**: 温暖米白背景 (#faf9f5) + 深棕文字 (#2b2621)
- **特点**: 优雅温暖、长时间阅读不累、书卷气息
- **强调色**: 橙棕色 (#bc6a3a)
- **适用**: 白天工作、写作、阅读文档

### 2. Claude-Like Dark (暗色主题) 🌙
- **文件名**: `mweb-claude-dark.mwebtheme`
- **配色**: 深棕背景 (#151210) + 米色文字 (#ddd4ca)
- **特点**: 低对比护眼、温暖深色、夜间友好
- **强调色**: 浅橙色 (#d59567)
- **适用**: 夜间编程、深夜写作、降低眼疲劳

### 3. Claude-Like Grey (灰色主题) 🌆
- **文件名**: `mweb-claude-grey.mwebtheme`
- **配色**: 中性深灰背景 (#2e343c) + 浅灰文字 (#e8eef5)
- **特点**: 专业现代、高对比度、全天候使用
- **强调色**: 暖橙灰色 (#d69870)
- **适用**: 技术文档、代码笔记、专业写作

## 安装方法

### MWeb 4
1. 打开 MWeb 偏好设置
2. 选择「自定义主题」
3. 点击「添加亮色主题」或「添加暗色主题」
4. 导入对应的 `.mwebtheme` 文件
5. 在编辑器中选择该主题

### 手动安装
将 `.mwebtheme` 文件复制到以下目录：
```
~/Library/Containers/com.coderforart.MWeb3/Data/Documents/themes/
```

## v2.0 版本亮点 🎯

### ✨ 全新优化
- **字体渲染升级**: 启用 OpenType 特性（连字、字偶距调整）
- **字重精细化**: 标题采用渐进式字重（650→600），层次更清晰
- **代码高亮增强**: 优化语法着色，关键词、函数、符号更易区分
- **列表样式改进**: Marker 颜色与主题色统一，视觉更协调
- **表格设计升级**: 表头背景、悬停效果、边框优化
- **脚注区域**: 新增独立脚注样式，分隔更清晰
- **移动端适配**: 响应式字号调整，手机阅读更舒适

### 🎨 字体配置（完整回退链）
- **正文**: Anthropic Serif → Source Serif 4 → Georgia → 思源宋体 → PingFang SC
- **代码**: SF Mono → Menlo → Cascadia Code → Consolas → Sarasa Mono SC
- **界面**: PingFang SC → Hiragino Sans GB → Noto Sans CJK SC

### 💻 代码高亮（精心调校）
所有主题都包含完整的语法高亮支持：
- **关键字**: 紫色系 + 中等字重（500），突出但不刺眼
- **字符串**: 绿色系，柔和舒适
- **数字/布尔**: 橙色系，醒目易识
- **注释**: 灰色系 + 斜体，降低干扰
- **属性/标签**: 蓝色系 + 中等字重（500），结构清晰
- **函数/类名**: 加粗（550），一眼识别
- **运算符/标点**: 中灰色，不喧宾夺主

### 📐 排版细节（像素级打磨）
- **行高**: 1.62（黄金比例，阅读舒适）
- **标题字号**: 1.88em → 1.52em → 1.26em → 1.12em（精心调校）
- **标题字重**: 650 → 630 → 620 → 610（渐进式，层次分明）
- **标题间距**: 字母间距 -0.01em（紧凑优雅）
- **代码块**: 圆角 6px，精细阴影，专业感
- **内联代码**: 圆角 4px，微妙高光，立体感
- **表格**: 表头加粗边框（1.5px），悬停淡色背景
- **引用块**: 2.5px 左边框，圆角 8px，背景色区分
- **链接**: 悬停显示下划线（渐变动画 0.15s）
- **任务列表**: 复选框主题色，指针悬停

### 响应式设计
- 最大宽度: 780px（标准）/ 820px（1400px+）/ 880px（1800px+）
- 移动端适配（768px 以下）
- 自适应内边距

## 设计理念

这些主题基于 Anthropic Claude 的设计语言，追求：
- **可读性**: 优化的行高、字号和颜色对比
- **舒适性**: 温暖的色调，减少视觉疲劳
- **一致性**: 统一的设计语言和视觉节奏
- **专业性**: 适合技术写作和长文档编辑

## MWeb 主题格式说明

`.mwebtheme` 文件由两部分组成：

### 1. JSON 配置（编辑器样式）
定义编辑器中的文本颜色和样式：
```json
{
  "h1": { "isBold": true, "color": "1c1815" },
  "editor": { "bgColor": "faf9f5", "color": "2b2621" },
  ...
}
```

### 2. CSS 样式（预览样式）
位于 `=============================================` 分隔符之后，定义预览和导出时的样式。

## 技术细节

- **颜色格式**: 6位十六进制（不带 `#`）
- **字体回退**: 多层次字体栈，确保跨平台兼容
- **语法高亮**: 基于 PrismJS 标准 token 类型
- **CSS Reset**: 包含标准化样式，确保一致显示
